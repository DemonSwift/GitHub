# GitHub

## GitHub入门

#### github是一个网站：https://github.com

#### git:版本控制系统  github是一个面向开源以及私有的软件项目托管平台

#### github上，可以接触最新最前沿的测试项目，了解最火热的IT方向，从而跟上技术

#### github上有很多自然语言代码编写的优质教学资料

#### github优势：只支持Git；完整协议支持；在线文件编辑；社交网络元素；特色工作模式；私有仓库托管；Ruby on Rails
